## 자바의 예외처리
### 예외 : 실행중 오류나 악영향을 미치는 상황
- 자바에서는 실행 중 발생하는 에러를 예외로 처리

### 실행중 예외가 발생하면 : 응용 프로그램이 예외 처리 하도록 호출
- 응용프로그램이 예외를 처리하지 않으면 프로그램 강제 종료

## 예외 처리문 try-catch-finaily문
```java
try {
			예외가 발생할 가능성이 있는 실행문(try블록)
		}
		catch(처리할 예외 타입 선언) {
			예외 처리문(catch블록)
		}
		finally {
			예외가 발생 여부와 상관없이 무조건 실행되는 문장(finally블록) 
            //생략가능
		}
```
### catch() 의 () 안에 예외타입은 int,double과간은 기본 타입은 사용 불가

## 조건문 switch

```java
switch(식){
    case 값1:
        실행 문장 //결과가 실행1일때
        break;
    case 값2:
        실행 문장 //결과가 실행2일때
        break;
    case 값3:
        실행 문장 //결과가 실행3일때
        break;
    default:
        실행문장 //어느 것과도 같지 않을 때
}
```

### 파일 추가
